About This Map


This interactive map shows the approximate locations of Christmas Tree farms across the Commonwealth of Kentucky.


Using This Map


Clicking/tapping an icon on the map shows shows detail.


The +/- buttons at the top left zoom in and out, as does the mouse wheel, and standard mobile gestures.


Data Sources


The primary data source is Christmas Tree Farm survey, supported by the Kentucky Horticulture Council.
